# **uman**

Uman -- programming language in Russian language for teaching children programming basics  

### Типы данных:
целое8
целое16
целое32
целое64

дробное32
дробное64

строка
символ
буль

### Переменные: 


### Синтаксис:
Создание текстовой переменной:
```
    [идентификатор]: [тип данных] = [значение];
    текст: строка = "Привет, мир!";
```
